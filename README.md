## Калькулятор комплексных чисел

### 1. Задание
Реализовать калькулятор, позволяющий складывать, вычитать, делить и умножать комплексные числа. Использовать язык программирования **Java**. Для реализации графического интерфейса использовать **Java Swing**. Добавить Unit тесты для проверки выполнения правильности математических операций. Калькулятор должен быть представлен в виде десктопного приложения для Windows 10.
